This is an entry for the Winter Melon Jam 2024.

https://itch.io/jam/wmj2024

Theme: Polarity

# Overview

2D Platformer where you use a magnet gun/device to create red/blue rays that interact with red/blue polarized surfaces/objects to reach the door at the end of each level.
